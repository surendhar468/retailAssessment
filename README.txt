# Retail Assessment

This application contains business logic for calculating bill invoice amount for retail store. 

This project contains business logic written using Java, and JUnit (Java) unit tests.

## Installation
1. Open the project by importing the Pom into an IDE, e.g. Eclipse, IntelliJ
2. Build the project
3. Navigate to the unit test file and run the JUnit tests directly or Run the test task of the Maven pom

## Test / Run
1. Open the project by importing the Pom into an IDE, e.g. Eclipse, IntelliJ
2. Go to test cases inside in src/test folder.
3. Run all test cases classes.
